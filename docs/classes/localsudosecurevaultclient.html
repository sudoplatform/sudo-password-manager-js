Class LocalSudoSecureVaultClient

An implementation of the SecureVaultClient interface that uses IndexedDB as its backing data store.

Note for NodeJS: To use this class from within NodeJS, you must first polyfill an indexeddb implemention, e.g.:

  require('fake-indexeddb/auto')
