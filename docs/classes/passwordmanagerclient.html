Class PasswordManagerClient

The PasswordManagerClient provides abstractions on top of SecureVaultService:

  1. Session management to persist SecureVaultService credentials.
  2. Interpretation of the blob from the SecureVaultService as a password vault and a nice interface to access/modify said vault.

It presents these abstractions in an interface similar to other Sudo Platform product SDKs, even though the underlying mechanisms differ.

PasswordManagerClient implementations must take great care to minimize the exposure of the user's master password and secret code.

    • Defined in src/passwordManagerClient.ts:517

    Unlocks the vaults. Operations that access vault data will succeed until lock is called.

    If the user's secret code is not already present on this device, then the user's secret code must be provided to this method or it will throw. The user's secret code will then be stored on this device for future use.
